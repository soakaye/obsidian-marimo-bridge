# Quickstart: Naming Marimo Temporary Files

## Introduction
This feature updates the plugin to avoid generating temporary marimo files inside the OS temporary folder. Clicking "New Notebook" inside the marimo dashboard now generates structured files within the vault.

## How to use

1. Enable the plugin and open the marimo home dashboard.
2. Click the "New Notebook" button.
3. Observe that a new tab opens, and the file `untitled_marimo.py` is created inside the active folder (or vault root).
4. Editing this file directly updates the local vault python file.
