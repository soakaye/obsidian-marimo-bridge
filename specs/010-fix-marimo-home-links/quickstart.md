# Quickstart: Fix Marimo Home Links

## Requirements
- Node.js installed.
- Obsidian Desktop installed.

## Setup & Run
1. Run `npm install` to restore dependencies.
2. Run `npm run build` to compile the TypeScript files and bundle the plugin.
3. Reload the plugin in Obsidian settings.

## Manual Test Flow
1. Open the Marimo Home dashboard in Obsidian.
2. Ensure you have some recent/running notebooks.
3. Click a notebook link inside the home view.
4. Verify it opens as a new Obsidian tab and NOT in your external system web browser.
