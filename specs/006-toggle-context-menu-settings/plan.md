# 実装計画: 設定によるコンテキストメニュー有効/無効トグル機能の追加

**ブランチ**: `006-toggle-context-menu-settings` | **作成日**: 2026-06-19 | **仕様書**: [spec.md](file:///C:/Users/asuzuki.SSL/Documents/obsidian/working/.obsidian/plugins/obsidian-marimo-bridge/specs/006-toggle-context-menu-settings/spec.md)

**インプット**: `/specs/006-toggle-context-menu-settings/spec.md` の機能仕様書

## 概要

本機能の目的は、ファイルエクスプローラ上での右クリックメニュー（コンテキストメニュー）に「Create new marimo notebook」を表示するかどうかを設定メニュー（Settings）から切り替え可能にすることです。
トグル設定 `showContextMenu` を導入し、有効な場合（デフォルト: `true`）のみメニューを表示し、無効な場合はメニュー項目を非表示にします。この設定の切り替えは、リロードなしで即時に反映されます。

## 技術的コンテキスト

**言語/version**: TypeScript 5.4+ / ES2022

**主要な依存関係**: Obsidian API (特に `Setting` トグル、プラグインデータ永続化)

**ストレージ**: Obsidian ローカルデータ (プラグイン設定の `saveData` / `loadData` を介した `showContextMenu` トグル状態の保存)

**テスト**: Obsidian デスクトップ環境における手動検証

**対象プラットフォーム**: デスクトップ (Obsidian Desktop のみ)

**プロジェクトタイプ**: desktop-app (Obsidian プラグイン)

**パフォーマンス目標**: 設定変更が即座（100ms未満）に以降のコンテキストメニューに反映されること

**制約事項**: 特になし
